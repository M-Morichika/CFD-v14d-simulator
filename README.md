## What's new in v14d (The Ontological Turn)

Unlike previous formulations, v14d introduces a closed feedback loop that makes hysteresis an emergent property of the system rather than an imposed condition.

CFD v14d marks a significant theoretical leap from previous versions by introducing **Dynamic Hysteresis generated by state-dependent feedback**, completely decoupling the model from specific clinical scenarios (e.g., anesthesia profiles) to focus on the pure nonlinear dynamics of consciousness and attention.

| Feature | Pre-v14 (Static) | v14d (Dynamic & Non-autonomous) |
| :--- | :--- | :--- |
| **Internal Metabolism ($\pi_{int}$)** | Static control parameter. | **Time-evolving dynamical variable.** Driven by external perturbation and positive feedback from Total Info $M$. |
| **Hysteresis** | Required manual "sweep" functions. | **Self-generated.** Emerges naturally from the closed loop $M \to \pi_{int} \to \rho \to M$. |
| **Extinction (Avalanche)** | Gradual fading. | Sudden structural collapse when $\pi_{int}$ drops below the ignition threshold $\lambda_0$. |
| **Recovery (Delay)** | Immediate recovery when perturbation stops. | **Crawling delay.** Recovery is stalled because $M=0$ provides no feedback $f(M)$ to pull $\pi_{int}$ back up. |
| **Self-inhibition** | None. | Added $-\kappa\rho^2$ to stabilize density growth and enforce capacity limits. |

## Equation System (v14d)

This simulator solves the following coupled non-autonomous system in real-time using finite difference methods (Upwind scheme for advection, 5-point Laplacian for diffusion).

**1. Evolution of density field $\rho(\beta, t)$** (Advection-Diffusion-Reaction)
$$\frac{\partial \rho}{\partial t} = -\nabla\cdot(\rho\mathbf{v}) + D_{mask}\nabla^2\rho + (\pi_{int}-\lambda_0)\rho - \kappa\rho^2$$

**2. Dynamics of Internal Metabolism $\pi_{int}(t)$** (Non-autonomous Loop)
$$\frac{d\pi_{int}}{dt} = -k_{drug}(\pi_{int} - \pi_{tgt}) + k_{rec}(\pi_{base} - \pi_{int}) + k_{fb}M$$

**3. Total Information $M$** (Order Parameter)
$$M(t) = \iint \rho(\beta, t) d\beta$$

**4. Flow velocity & defensive viscosity**
$$\mathbf{v} = -\frac{B}{\gamma(G)}\nabla F, \quad \gamma(G) = \gamma_{min} + \frac{\gamma_{max}-\gamma_{min}}{1+e^{-a(G-G_{th})}}$$

**5. Transformation resistance $G$**
$$G[\rho,F] = k_G \iint \rho F r dr d\theta$$

**6. Dynamic Potential $F$ (Erosion & Weathering)**
$$\frac{\tau_{base} R / (r+r_0)}{1+\rho/\rho_0} \frac{\partial F}{\partial t} = -(F-F_0) - \int W(\beta,\beta')\rho(\beta') d\beta'$$

## Open Mathematical Structure

The CFD v14d system defines a coupled nonlinear, non-autonomous PDE with state-dependent feedback:

$$M \to \pi_{int}(t) \to \rho(\beta,t) \to M$$

This feedback loop dynamically shifts the bifurcation structure of the system and produces hysteresis without external control. We believe this formulation represents a non-standard class of reaction–diffusion–advection systems with the following analytical challenges:

* Existence and stability of localized solutions ($\beta^*$)
* Dynamic bifurcation crossing under time-evolving control parameter $\pi_{int}(t)$
* Analytical characterization of hysteresis loops in closed-loop systems
* Role of state-dependent viscosity $\gamma(G)$ in phase-transition-like behavior
* Well-posedness under evolving potential $F(\beta,t)$

The simulator provides a concrete numerical realization of this system.
